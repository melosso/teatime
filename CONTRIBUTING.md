# Contributing to Teatime

First off, thanks for taking the time to contribute! Please submit a PR with your suggested changes.

#### Building

Requires the [.NET 11 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/11.0).

```bash
git clone https://github.com/melosso/teatime.git
cd teatime/Teatime
dotnet restore
dotnet watch --project src/Teatime
```

Browse to `http://localhost:5000` to visit the site.
