Markdown is the format that gets out of your way. It is readable as source, it diffs cleanly in git, and it will still open in any text editor in twenty years.

## Tables work too

| Feature   | Supported |
| --------- | --------- |
| Tables    | Yes       |
| Footnotes | Yes       |
| Code      | Yes       |

Write once, render anywhere.
