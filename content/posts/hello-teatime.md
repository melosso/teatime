Welcome to **Teatime** — a personal blog engine that serves Markdown straight from a folder. No database, no build step, no JavaScript framework.

## Why

Because writing should be as simple as dropping a `.md` file into `content/posts/`.

```csharp
Console.WriteLine("Hello, Teatime");
```

That is the whole idea.
